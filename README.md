# Накруктака метрики (cheating-metrica)
![GitHub Release](https://img.shields.io/github/v/release/korzyno87/cheating-metrica)   ![GitHub User's stars](https://img.shields.io/github/stars/korzyno87)       

## **Описание**
*Данная программка позваляет посещать сайты по списку, описанному в файле.* 

Для изменения определения 'устройства' используються библиотека **fakeuseragent**, который подменяет информацию о подключаемом устройстве в случайном порядке. 

Для изменения ip адреса с которого устройство посещало данный сайт используеться прокси. В случии успешного тестирования будет выбор того или иного прокси, но на сегодняшний день лучшую работу поакзала библиотека **proxybroker**

Так же длительностью пребывания на сайте можно управлять задав время ожидания на сайте в секиндах, по умолчанию 2 секунды. (Чем больше время пребывания тем меньше посещений в день) Так же есть параметр управлением времени успешной загрузки сайта

При удачном посещении сайта (статус код :200) программа сохраняет скрин посещенного сайта с указанием адреса и времени посещения

## **Используемые технологии**
* 'Python 3.11.3'
* 'selenium'

### Установка дополнительных библиотек
```bash
    pip install selenium
```


## **Схема работы**
1. Формирование списка proxy
2. Выбор user-agent
3. Подмена агента и проксирование для открытия сайтов из списка с регулируемым временем пребывания на сайте
4. В случии удачного посещения запись скрина сайта в лог
