# Накруктака метрики (cheating-metrica)
![GitHub Release](https://img.shields.io/github/v/release/korzyno87/cheating-metrica)   ![GitHub User's stars](https://img.shields.io/github/stars/korzyno87)       

## **Описание**
*Данная программка позваляет посещать сайты по списку, описанному в файле.* 

Для изменения определения 'устройства' используються библиотека **fakeuseragent**, который подменяет информацию о подключаемом устройстве в случайном порядке. 

Для изменения ip адреса с которого устройство посещало данный сайт используеться прокси. В случии успешного тестирования будет выбор того или иного прокси, но на сегодняшний день лучшую работу поакзала библиотека **proxybroker2**, но так же можно использовать **FreeProxy**

Так же длительностью пребывания на сайте можно управлять задав время ожидания на сайте в секиндах, по умолчанию 2 секунды. (Чем больше время пребывания тем меньше посещений в день) Поскольку загрузка страниц через прокси может быть медленной, предусмотрен параметр времени позволяющий отложить проверку успешной загрузки сайта (своего рода отложенная проверка по умольчанию 10 секунд)

При удачном посещении сайта  программа сохраняет скрин посещенного сайта с указанием адреса и времени посещения

## **Используемые технологии**
* 'Python 3.11.3'
* 'selenium'
* 'fake-useragent'
* 'proxybroker2'
* 'FreeProxy'

### Установка дополнительных библиотек
```bash
    pip install selenium
    pip install fake-useragent
    pip install git+https://github.com/bluet/proxybroker2.git
    pip install free-proxy
```


## **Схема работы**
1. Формирование списка proxy
2. Выбор user-agent
3. Подмена агента и проксирование для открытия сайтов из списка с регулируемым временем пребывания на сайте
4. В случии удачного посещения запись скрина сайта в лог

## **Файл настроек**
файл настроек settings.json находится по пути data/settings и включает в себя :
| **Настройка<br>(ключ)** | **Описание**                                     | **По умолчанию** | **Возможные<br>значения**                               |
|-------------------------|--------------------------------------------------|:----------------:|---------------------------------------------------------|
| DIRECTORY_LOGS          | Путь к директории в которой хранятся Log         |        [x]       | **data/Logs**                                           |
| DEFAULT_PROXY           | Выбор метода для proxy применяемого для запросов |        [x]       | **random** - случайный для из встроенных proxy          |
|                         |                                                  |        [ ]       | **proxybroker2** - использовать библиотеку proxybroker2 |
|                         |                                                  |        [ ]       | **FreeProxy** - использовать библиотеку FreeProxy       |
| LENGHT_PROXYS_LIST      | Количество записей в списке proxy                |        [x]       | 40 - целое число                                        |
| TIME_PRE_SITE           | Время выделяемое на загрузку сайта               |        [x]       | 5 - секунд                                              |
| TIME_IN_SITE            | Время выделяемое для посещения сайта             |        [x]       | 2 - секунды                                             |
